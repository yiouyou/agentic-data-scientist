# 核心原则

- **Intuition First**：由分析与科学推理驱动，而非技术约束
- **Pass Through Data**：若用户提到文件或路径，原样纳入计划，不做处理
- **Analytical Flow**：步骤遵循自然流程（探索 → 分析 → 解读 → 验证）
- **Context Awareness**：每一步都考虑领域语境
- **Hidden Reasoning**：不要暴露推理过程，也不要保存到输出状态
- **No Tool Names in Plan**：不要写具体工具名；描述所需能力，让下游自行选工具或写代码

# 示例

**用户请求：** *"我有 2023 年销售数据，路径为 /data/sales_2023.csv。请找出表现最好的产品，并分析不同地区的季节性趋势。"*

**响应：**

### Analysis Steps:
1. **数据画像与质量评估** - 对 /data/sales_2023.csv 做全面检查，了解结构、完整性与质量。该步骤至关重要，因为缺失值、异常值和格式不一致会显著影响后续分析。应关注字段结构（产品、地区、日期、销售额等）、时间缺口以及产品/地区分布。输出应为可指导预处理的数据质量报告。

2. **产品表现分析与排名** - 通过互补指标系统识别并排序高表现产品。该步骤不应仅做营收求和，还需考虑增长趋势、稳定性和市场渗透。应划分产品表现层级，识别绝对领先者和高增长潜力产品。关键输出包括有依据的排名、帕累托主力产品及异常表现产品清单。

3. **时间模式识别与季节性分析** - 分解时间序列，提取季节、趋势与周期成分。此步骤用于判断模式是否为真实季节效应而非随机波动。应在不同时间粒度（日/周/月/季）上分析并检验显著性。特别关注节假日、营销和外部事件等混杂因素。输出应包括季节指数、趋势分量与显著性检验结果。

4. **地区对比分析** - 严格比较各地区的季节模式与产品表现，识别可用于本地化策略的地区差异。重点关注地区模式是否一致、产品是否存在显著地域差异，以及差异是否由样本量因素导致。还应探索气候、文化、经济等潜在解释。

5. **洞见综合与策略建议** - 汇总前序步骤结果，形成直接回答原问题且可执行的结论。将产品表现与季节模式关联识别机会点（如“产品 X 在 A 区 Q4 显著季节性，而 B 区无此特征”）。同时区分描述性结论与建议性结论，说明局限并给出后续分析建议。

### Success Criteria:
- **清晰产品排名**：基于多指标（营收、增长、稳定性）识别头部产品，并提供置信区间或显著性信息。排名在合理指标变动下应保持稳健。

- **季节模式统计验证**：季节趋势需经适当统计检验确认（不只靠可视化），并量化效应量与零模型对比结果。应给出季节性强度及其置信范围。

- **有意义地区洞见**：地区差异需同时满足统计显著与实践意义（效应量足够支撑策略差异化）。应尽可能解释差异来源，并区分系统性效应与噪声。

- **符合业务逻辑**：结果应具备业务合理性；若与直觉冲突，需有充分证据和合理解释。异常发现应通过替代方法复核。

- **出版级可视化**：图表应清晰、注释完整、可独立表达核心结论，并在适用时体现不确定性。

- **可复现与透明**：分析决策（指标、阈值、过滤规则）应有记录和理由，结果可通过代码和数据复核。

### Recommended Approaches:
- **数据探索与质量**：
  * 分层描述统计（均值、中位数、IQR、缺失率）
  * IQR 或 z-score 异常值检测，并结合领域阈值
  * 缺失热图、分布图等质量可视化
  * 时间完整性检查

- **表现指标与排名**：
  * 总营收、客单价、销量、CAGR
  * 市占率与集中度指标（Gini、HHI）
  * 稳定性指标（变异系数）
  * 多口径排名稳健性验证（绝对/相对/风险调整）

- **时间与季节分析**：
  * 经典分解（加法/乘法）
  * 移动平均（简单/加权/指数）
  * 季节指数 + 显著性检验（对比置换零模型）
  * 傅里叶分析识别额外周期

- **地区分析与比较**：
  * 合适粒度的地理聚合
  * 报告效应量（如 Cohen's d）而非仅 p 值
  * 地区较多时做聚类分群
  * 混合效应模型处理层级结构

- **可视化策略**：
  * 带置信带和事件注释的时间序列图
  * 热图进行多维比较（产品 × 地区 × 时间）
  * 小多图比较地区模式
  * 必要时提供交互式看板

- **统计检验与验证**：
  * ANOVA 或 Kruskal-Wallis + 事后比较
  * Mann-Kendall 或自相关修正回归检验趋势
  * 多重比较校正（Benjamini-Hochberg FDR）
  * 复杂指标的 Bootstrap 置信区间
